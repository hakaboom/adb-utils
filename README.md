# adb-utils
![PyPI - Downloads](https://img.shields.io/pypi/dm/adb-utils)

对adb常用接口的二次开发

## Requirements
- python>=3.8
- 项目里用到了[base_image](https://github.com/hakaboom/base_image) 因此需要额外安装opencv-python


## Installation
pip3 install adb-utils
